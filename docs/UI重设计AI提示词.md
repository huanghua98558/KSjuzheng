# 📋 完整提示词 (以下整块复制)

```markdown
你是一个资深 UI/UX 设计师, 擅长桌面商业软件界面设计. 请为下面这个产品设计完整的用户界面.

## 一、产品背景

**产品名**: KS 短剧矩阵 (KS Drama Matrix)
**产品类型**: Windows 桌面软件 (PyQt6 原生 / 或 Electron 壳)
**面向用户**: 做快手短剧 CPS 分成的个人运营者 / 小工作室
**用户规模**: 管理 10-500 个快手账号
**核心价值**: 一键自动化 — 装了就能开始挣钱

**竞品参考**:
- KS184 (同类快手矩阵工具, 卖 ¥2000-5000)
- 一些抖音/快手数据工具 (¥1500-3000)
- 对标视觉: KS184 的 PyQt 外观, 但更现代化

## 二、目标用户画像

**核心用户**:
- 年龄: 25-40 岁
- 技术水平: 会用 Windows 软件, 不会 Python / 不看日志
- 背景: 做短剧 CPS 分成, 可能有 10-500 账号
- 痛点: 手动发布太累 / 不知道跟哪部剧 / 账号多了不好管
- 期望: 装了就能跑 / 不用改配置 / 出问题有提示

**设计取舍**:
- ✅ 大按钮大字 (不是密密麻麻的表格)
- ✅ 图标辅助 (不只是文字)
- ✅ 状态可视化 (圆点 / 进度条 / 颜色)
- ✅ 中文术语 (不露英文代码)
- ❌ 不要技术细节 (CPU / 内存 / 日志不在首页)
- ❌ 不要一屏塞 20 个功能
- ❌ 不要让用户决定 "选哪个算法"

## 三、设计风格要求

### 视觉风格
- **主色调**: 深色专业系 (深蓝/深紫/深灰为主, 配橙黄或青色高亮)
  - 原因: 夜间使用友好 + 对标 KS184 专业感
- **布局**: 左侧侧边栏导航 + 中间主内容 + 右侧状态面板
- **卡片式**: 所有数据用卡片, 不用密集表格
- **图标**: Material Icons / Lucide Icons / Phosphor Icons (现代扁平)
- **字体**: 中文用 "PingFang SC" / "Microsoft YaHei", 英文用 "Inter" / "SF Pro"
- **动效**: 轻微过渡 (不要花哨), 加载 skeleton 不用 spinner

### 交互原则
- **渐进式披露**: 新手看到 5 个按钮, 高级用户点"⚙️ 高级"看到 20 个
- **即时反馈**: 按钮点击立即响应 (loading state 内嵌按钮内, 不弹新窗)
- **错误友好**: 不说 "Error: TypeError", 说 "网络不太好, 点这里重试"
- **撤销机制**: 危险操作 (删账号/批量冻结) 3 秒撤销
- **键盘友好**: F1 帮助 / Ctrl+S 保存 / Esc 关弹窗

### 不做的
- ❌ 不要 Material Design 纯风格 (太像网页)
- ❌ 不要 iOS 圆角满 (桌面软件不是 App)
- ❌ 不要纯黑 (AMOLED 风), 用深灰 #1a1a1a
- ❌ 不要过饱和色 (对运营者 10 小时使用疲劳)

## 七、核心交互流程

### 流程 1: 新用户首次启动 → 发第一条视频
```
启动 → 引导 Wizard (5 步) → 激活 → 扫码加第一个账号 
  → 等 MCN 验证 → ⚙️ 默认设置好 → 点"立即发布" 
  → 看到第一个任务运行 → 完成 → 收益页看到 (即使 ¥0.01)
  → 🎉 成功动画 + 引导 "继续加号"
```

### 流程 2: 日常使用
```
早上打开 → 仪表盘 (昨日挣了 ¥23) → 
  点"立即发布" (或 autopilot 已在跑) → 
  切到"爆款雷达" 看今天推了什么新剧 → 
  看看 [AI 建议] 有没有 → 
  关掉 (托盘继续跑)

晚上再打开 → 仪表盘 (今天 ¥15, 少了) → 
  "诊断" 看为啥 → 发现 2 账号冻结 →
  点"修复" → 跟客服反映 → 等明天
```

### 流程 3: 升级付费
```
免费试用到第 5 天 → 看到"续费"悬浮条 → 
  点"升级" → 看到 3 档对比 → 选专业版 →
  微信扫码付款 ¥399 → 激活码自动填入 → 
  解锁所有云服务 → 🎉 升级成功 → 
  首页顶部紫色横条 (满足感)
```

### 流程 4: 出现故障
```
用户回家发现: ⚠️ 3 账号显示"冷却中" →
  点"账号" → 看到黄色卡片 "cookie 过期, 需重新扫码" →
  一键扫码 → 恢复 → 
  (技术细节用户看不到)
